Create directory `rtl/` if it does not exist. Create `rtl/divider.v` implementing a parameterized restoring binary division module.

**Module declaration (ANSI style):**
```
module divider #(
    parameter WIDTH = 27
) (
    input  wire             clk,
    input  wire             rst,
    input  wire             start,
    input  wire [WIDTH-1:0] dividend,
    input  wire [WIDTH-1:0] divisor,
    output reg              busy_ff,
    output reg              done_ff,
    output reg              dbz_ff,
    output reg  [WIDTH-1:0] quotient_ff,
    output reg  [WIDTH-1:0] remainder_ff
);
```

**Internal registers (all with _ff suffix per INFRA-03):**
- `dividend_copy_ff` [WIDTH-1:0] -- latched dividend shifted left each iteration
- `divisor_copy_ff` [WIDTH-1:0] -- latched divisor (constant during operation)
- `acc_ff` [WIDTH-1:0] -- accumulator for restoring division
- `quo_ff` [WIDTH-1:0] -- quotient being assembled bit by bit
- `iter_ff` [$clog2(WIDTH)-1:0] -- iteration counter (0 to WIDTH-1, so 5 bits for WIDTH=27)

**Combinational wire:**
```
wire [WIDTH:0] acc_next;
assign acc_next = {acc_ff[WIDTH-2:0], dividend_copy_ff[WIDTH-1]} - {1'b0, divisor_copy_ff};
```
This is the trial subtraction: shift accumulator left by 1, bring in MSB of dividend_copy_ff, subtract divisor. The extra bit (WIDTH+1 total) captures the borrow.

**Sequential block (`always @(posedge clk)`) -- non-blocking only:**
- On `rst`: clear all _ff registers to zero, `busy_ff <= 0`, `done_ff <= 0`, `dbz_ff <= 0`.
- On `start` (and not busy):
  - If `divisor == 0`: set `dbz_ff <= 1`, `done_ff <= 1` (divide-by-zero).
  - Else: latch `dividend_copy_ff <= dividend`, `divisor_copy_ff <= divisor`, clear `acc_ff`, `quo_ff`, `iter_ff` to 0, set `busy_ff <= 1`.
- When `busy_ff` is high (iterating):
  - Default `done_ff <= 1'b0` at top of clocked block.
  - If `acc_next[WIDTH] == 1'b0` (no borrow -- subtraction succeeded): `acc_ff <= acc_next[WIDTH-1:0]`, `quo_ff <= {quo_ff[WIDTH-2:0], 1'b1}`.
  - Else (borrow -- subtraction failed, restore): `acc_ff <= {acc_ff[WIDTH-2:0], dividend_copy_ff[WIDTH-1]}`, `quo_ff <= {quo_ff[WIDTH-2:0], 1'b0}`.
  - Always shift dividend: `dividend_copy_ff <= {dividend_copy_ff[WIDTH-2:0], 1'b0}`.
  - Increment: `iter_ff <= iter_ff + 1`.
  - When `iter_ff == WIDTH - 1`: set `busy_ff <= 0`, `done_ff <= 1`, `quotient_ff <= {quo_ff[WIDTH-2:0], final_bit}`, `remainder_ff <= final_acc`. The final quotient bit and accumulator value come from the last iteration's acc_next check (same logic as above, applied on the final cycle).

**CRITICAL implementation details:**
- `done_ff` must pulse HIGH for exactly ONE clock cycle, then return to 0. Set `done_ff <= 1'b0` as the default at the top of the sequential block's else branch (non-reset path), then override to 1 only on the completion cycle and the dbz cycle.
- The module takes exactly WIDTH clock cycles to complete a division (plus 1 cycle for start latch).
- No `for` loops anywhere (INFRA-05).
- No `always @(*)` block needed if all logic is purely sequential. However, the `acc_next` wire IS combinational (assign statement), which is fine -- it is not inside an always block.
- Every `if` chain must have a final `else` (INFRA-07). Every `case` must have `default:` (INFRA-07).
- Blocking (`=`) must NOT appear in the `always @(posedge clk)` block (INFRA-04).

  <action>
Create `rtl/prime_engine.v` implementing the 6k+/-1 trial division FSM.

**Module declaration (ANSI style):**
```
module prime_engine #(
    parameter WIDTH = 27
) (
    input  wire             clk,
    input  wire             rst,
    input  wire             start,
    input  wire [WIDTH-1:0] candidate,
    output wire             done_ff,
    output wire             is_prime_ff,
    output wire             busy_ff
);
```
Note: `done_ff`, `is_prime_ff`, and `busy_ff` are output wires driven from internal registers. Declare internal `reg` versions with _ff suffix and assign to the output wires via `assign` or declare outputs as `output reg`.

**State encoding -- use localparam with explicit 3-bit encoding:**
```
localparam [2:0]
    IDLE      = 3'd0,
    CHECK_2_3 = 3'd1,
    INIT_K    = 3'd2,
    TEST_KM1  = 3'd3,
    TEST_KP1  = 3'd4,
    DONE      = 3'd5;
```

**Internal _ff registers:**
- `state_ff` [2:0] -- current FSM state
- `candidate_ff` [WIDTH-1:0] -- latched candidate value
- `d_ff` [WIDTH-1:0] -- current trial divisor
- `k_ff` [WIDTH-1:0] -- current k value (d = 6k-1 or 6k+1)
- `is_prime_result_ff` -- latched primality result
- `done_out_ff` -- latched done signal
- `next_d_ff` [WIDTH-1:0] -- registered next divisor value
- `div_start_ff` -- registered divider start pulse (driven from combinational logic, latched into FF)

**Combinational signals (no _ff suffix, driven in always @(*)):**
- `next_state` [2:0]
- `next_candidate` [WIDTH-1:0]
- `next_d` [WIDTH-1:0]
- `next_k` [WIDTH-1:0]
- `next_is_prime_result`
- `next_done_out`
- `div_start` -- combinational start pulse to divider

**Bound check -- combinational wire (DSP48 inferred):**
```
wire [2*WIDTH-1:0] d_squared;
assign d_squared = d_ff * d_ff;
wire bound_exceeded = (d_squared > {{WIDTH{1'b0}}, candidate_ff});
```
Use strict greater-than (`>`). When `d_ff * d_ff > candidate_ff`, all remaining divisors are too large; candidate is prime.

**Divider instance:**
```
wire div_done;
wire [WIDTH-1:0] div_remainder;
wire div_busy;

divider #(.WIDTH(WIDTH)) u_div (
    .clk          (clk),
    .rst          (rst),
    .start        (div_start_ff),
    .dividend     (candidate_ff),
    .divisor      (d_ff),
    .busy_ff      (div_busy),
    .done_ff      (div_done),
    .dbz_ff       (),           // unused -- divisor is never 0 in our usage
    .quotient_ff  (),           // unused -- we only need remainder
    .remainder_ff (div_remainder)
);
```

**Sequential block (`always @(posedge clk)`) -- non-blocking only (INFRA-04):**
```
always @(posedge clk) begin
    if (rst) begin
        state_ff           <= IDLE;
        candidate_ff       <= {WIDTH{1'b0}};
        d_ff               <= {WIDTH{1'b0}};
        k_ff               <= {WIDTH{1'b0}};
        is_prime_result_ff <= 1'b0;
        done_out_ff        <= 1'b0;
        div_start_ff       <= 1'b0;
    end else begin
        state_ff           <= next_state;
        candidate_ff       <= next_candidate;
        d_ff               <= next_d;
        k_ff               <= next_k;
        is_prime_result_ff <= next_is_prime_result;
        done_out_ff        <= next_done_out;
        div_start_ff       <= div_start;
    end
end
```

**Combinational block (`always @(*)`) -- blocking only (INFRA-04):**
Default ALL next_* signals to their current _ff values at the top. Default `div_start = 1'b0`.

State logic:

- **IDLE:** If `start`, transition to CHECK_2_3 and latch `next_candidate = candidate`. Else stay IDLE.

- **CHECK_2_3:**
  - If `candidate_ff <= 1`: NOT prime, go DONE with `next_is_prime_result = 0`.
  - If `candidate_ff == 2` or `candidate_ff == 3`: IS prime, go DONE with `next_is_prime_result = 1`.
  - If `candidate_ff[0] == 0` (even): NOT prime, go DONE with `next_is_prime_result = 0`.
  - Else: Need to check divisibility by 3. Set `next_d = 3`, `div_start = 1`, go to a sub-state.

  **Divisibility by 3 approach:** Add two extra states for the div-by-3 check:
  ```
  localparam [2:0]
      IDLE       = 3'd0,
      CHECK_2_3  = 3'd1,
      WAIT_DIV3  = 3'd2,
      INIT_K     = 3'd3,
      TEST_KM1   = 3'd4,
      TEST_KP1   = 3'd5,
      DONE       = 3'd6;
  ```
  Update state encoding to 3 bits (7 states, fits in 3 bits). In CHECK_2_3, for odd candidates > 3: set `next_d = 3`, `div_start = 1`, go to WAIT_DIV3.

- **WAIT_DIV3:** Wait for `div_done`. When done: if `div_remainder == 0`, NOT prime (go DONE). Else set `next_k = 1`, `next_d = 5` (first 6k-1 divisor), go INIT_K.

- **INIT_K:** Check bound: if `bound_exceeded` (d_ff*d_ff > candidate_ff), IS prime (go DONE). Else fire divider: `div_start = 1`, go TEST_KM1.

- **TEST_KM1:** Wait for `div_done`. When done:
  - If `div_remainder == 0`: NOT prime, go DONE.
  - Else: set `next_d = d_ff + 2` (switch from 6k-1 to 6k+1), `div_start = 1`, go TEST_KP1.

- **TEST_KP1:** Wait for `div_done`. When done:
  - If `div_remainder == 0`: NOT prime, go DONE.
  - Else: set `next_k = k_ff + 1`, `next_d = d_ff + 4` (advance to next 6(k+1)-1 = current 6k+1 + 4), go INIT_K.

- **DONE:** Assert `next_done_out = 1`. On next cycle, return to IDLE (or stay in DONE until a new `start`). Set `next_done_out = 0` and go IDLE unconditionally (done_ff pulses for one cycle).

**CRITICAL details:**
- `div_start` is a COMBINATIONAL signal fed to `div_start_ff` which is the REGISTERED version going to the divider. This means the divider sees `start` one cycle after the FSM decides to fire it. Account for this: the FSM transitions to the WAIT state on the SAME cycle it sets `div_start = 1`. The divider latches inputs on the next posedge when `div_start_ff` goes high.
- ALTERNATIVE (simpler): Make `div_start` combinational and wire it DIRECTLY to the divider's `start` port (not through a FF). Then the divider starts on the same cycle. This avoids the one-cycle delay. Choose this approach: wire `div_start` (combinational) directly to `u_div.start`.
- `done_out_ff` pulses for exactly ONE cycle. In the DONE state's combinational logic, set `next_done_out = 1` on entry, then the next cycle transitions to IDLE with `next_done_out = 0`.
- Actually, since `done_out_ff` is registered: when the FSM decides `next_state = DONE` and `next_done_out = 1`, those values appear on the next posedge. So `done_out_ff` goes high for one cycle (the cycle the FSM is IN the DONE state). The combinational logic for DONE state sets `next_done_out = 0` and `next_state = IDLE`, so the FOLLOWING cycle done_out_ff goes low and state returns to IDLE.
- Every `if` has a final `else` (INFRA-07). Every `case` has `default:` (INFRA-07).
- No `for` loops (INFRA-05).
- `d_squared` comparison uses `>` (strict), NOT `>=`. This is critical: for candidate=25, d=5, d*d=25 which is NOT > 25, so the loop continues and tests 25%5==0 (composite). Correct.

**Output assignments:**
```
assign done_ff = done_out_ff;
assign is_prime_ff = is_prime_result_ff;
assign busy_ff = (state_ff != IDLE) && (state_ff != DONE);
```
